Demo Overview

This demonstration shows the difference between:

  • Legacy Image: Old Alpine 3.14 base with vulnerable Apache packages
  • WizOS Image: WizOS Alpine 3.19 base with minimal vulnerability Apache from Wiz apk packages

Key Differences

  • Legacy uses alpine:3.14 with outdated packages containing known CVEs
  • WizOS uses registry.os.wiz.io/alpine:3.19 with continuously updated, minimal vulnerability packages
  • Both serve the same Apache web application
  • WizOS packages are authenticated and verified through Wiz's apk-auth mechanism

How to Test

  1. Scan legacy image: make scan-apache-legacy
  2. Scan WizOS image: make scan-apache-wizos
  3. Compare vulnerability counts
